# 虛擬試穿 API 服務清單

## 專業虛擬試穿 API 服務

### 1. Replicate API ⭐⭐⭐⭐⭐（最推薦）

**網站：** https://replicate.com

**特點：**
- 提供多種虛擬試穿模型（IDM-VTON、OOTDiffusion 等）
- 按使用次數付費，價格透明
- 不需要自己搭建模型
- 有免費額度試用

**價格：**
- 約 $0.01-0.05/次
- 免費額度：通常有 $5-10

**優點：**
- ✅ 效果最好（專門的虛擬試穿模型）
- ✅ 價格便宜
- ✅ 設置簡單
- ✅ 多種模型可選

**缺點：**
- ⚠️ 需要網路連接
- ⚠️ 每次處理需要 10-30 秒

**適合：**
- 個人開發者
- 小到中型項目
- 想要最好效果

## 快速開始

### 使用 Replicate（推薦）：
1. 註冊：https://replicate.com/signup
2. 獲取 Token：https://replicate.com/account/api-tokens
3. 設置：在 `.env.local` 添加 `REPLICATE_API_TOKEN=你的token`
4. 完成！

### 使用 OpenAI：
1. 獲取 Key：https://platform.openai.com/api-keys
2. 設置：在 `.env.local` 添加 `OPENAI_API_KEY=你的key`
3. 完成！
