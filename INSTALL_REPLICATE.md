# 安裝 Replicate 套件

## 📋 步驟 1：安裝 replicate 套件

在終端執行以下命令：

```bash
cd /Users/minyu/closet
npm install replicate
```

## 📋 步驟 2：確認環境變數

確保 `.env.local` 文件中有 `REPLICATE_API_TOKEN`：

```
REPLICATE_API_TOKEN=r8_你的token
```

## 📋 步驟 3：重啟開發服務器

安裝完成後，重啟開發服務器：

```bash
npm run dev
```

## ✅ 完成

安裝完成後，就可以在測試頁面使用 Replicate 攤平功能了！
